# Examples

These example layouts mirror the layered setup used in `README.md` and the bug-report form. They are intentionally small and generic.

- `examples/personal-dotfiles/` is the base repo.
- `examples/work-dotfiles/` is the overlay repo that extends the base.

You can copy either directory as a starting point, or use them as a minimal repro shape when you are debugging `dotty` itself.
